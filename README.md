# YMM4素材同梱プラグイン

YMM4素材同梱プラグインは、YukkuriMovieMaker のプロジェクト素材をまとめて管理・共有するためのプラグインです。  
特定のファイルを除外したり、素材をパッケージ化して他のユーザーと同じ環境で編集を再開できます。

---

## 概要

- YukkuriMovieMaker プロジェクト内の素材を簡単にまとめたり展開可能  
- 除外設定ウィンドウで不要なファイルを選択的に除外可能  
- YukkuriMovieMaker のテーマに連動（将来的に完全対応予定）  

---

## 主な機能

### プロジェクト素材のパッケージ化

- プロジェクト内の素材を一括でパッケージ化  
- 他のユーザーと共有可能  
- 除外設定で指定したファイルはパッケージ化から除外されます  

### 除外設定

- パッケージ化したくないファイルをチェックボックスで簡単に除外  
- 「すべて選択」「すべて解除」ボタンで一括操作  
- OK で設定を保存、キャンセルで変更を破棄  

---

## インストール方法

1. リリースから最新の `.ymme` をダウンロード  
2. ダウンロードした `.ymme` を起動してインストール  
3. ツールから「素材同梱プラグイン」を起動  

---

## 使用方法

1. YukkuriMovieMaker を起動  
2. プラグインメニューから「素材同梱プラグイン」を選択  
3. 除外設定ウィンドウを開き、除外したいファイルにチェック  
4. OK ボタンで保存  
5. 必要に応じてプロジェクト全体をパッケージ化  

---

## ymmpx ファイルの関連付け

- `.ymmpx` ファイルをダブルクリックで自動的に展開できるように、  
  「ファイル関連付け」機能を利用できます。  
- プラグインのメイン画面で **「ymmpx関連付け」ボタン** をクリックすると、  
  `YMMResourceUnpackerApp` が管理者権限で起動し、自動的に関連付けを設定します。  
- 設定後は `.ymmpx` ファイルを開くだけで、  
  YukkuriMovieMaker 用プロジェクトと素材が展開されます。
